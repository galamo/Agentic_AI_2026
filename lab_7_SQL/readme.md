# Whats next?

1. run `docker compose up`
2. run the scripts from package.json

- "index-schema": "node scripts/index-schema.js",
- "init-db": "node scripts/load-schema.js && node scripts/load-data.js"
- Open pgAdmin - check the data exist

3. cd client
4. run `npm i`
5. run `npm run dev`
6. lunch the client in the relevant url.
